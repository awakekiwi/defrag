# DEFRAG

A Lode Runner-inspired glitch-cyberpunk platformer. You're a stray process loose in a decaying mainframe — recover memory fragments, dodge garbage collectors, and dig memory holes to trap (or be trapped by) the machine.

## Play

Open `index.html` or visit the live GitHub Pages URL.

**Controls**
- Move: Arrow keys
- Climb: Up / Down on a conduit
- Dig: Z (left) / X (right)
- Zoom toggle: Tab

## Level Editor

Open `editor.html` to design your own sectors. Paint tiles, validate solvability, save, and export as JSON.

To play custom levels: export from the editor, then use **LOAD LEVELS** on the game's title screen to import the JSON file.

## Files

- `index.html` — the game
- `editor.html` — the level editor
